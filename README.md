This code is for compression and decompression of any file type using LZ77 algorithm for high compression and huffman encoding for faster compression.

File contents:

compress.cpp - Core LZ77 implementation
decompress.cpp - Core LZ77 implementation but of decompression
main.c - Simple Command Line Interface
Makefile - To create executables
pg1184 - Example text files

How to run:

```git clone https://github.com/Aryan-s08/File-compression-and-decompression.git```<br/>
```make```<br/>
```./main```<br/>
